# Golam Kibria — Personal Portfolio

A fully responsive personal portfolio website built with React, Vite, Tailwind CSS, and Firebase — showcasing my journey, skills, and projects as a Web Developer.

**🔗 Live Site:** [golam-kibria-prortfolio.web.app](https://golam-kibria-prortfolio.web.app)

## ✨ Features

- **Responsive Navbar** — smooth-scroll navigation to every section, with a mobile-friendly dropdown menu
- **Hero Section** — designation, intro, and a one-click **Resume Download** button
- **Social Links** — direct GitHub and LinkedIn buttons
- **About Me** — my programming journey, what I love building, and life outside of code
- **Technical Skills** — categorized by Frontend, Backend, Database, and Tools, with a "Learning" badge for skills currently in progress
- **Education** — academic background in a clean table layout
- **Projects** — at least 3 featured projects, each with a **View More** button leading to a detailed page containing:
  - Tech stack used
  - Project description
  - Live project link
  - Client-side GitHub repository link
  - Challenges faced during development
  - Planned future improvements
- **Contact Section** — Email, Phone, WhatsApp, and a working contact form (powered by EmailJS)
- **Footer** — quick links and social icons

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, DaisyUI
- **Routing:** React Router (with dynamic project detail routes)
- **Email:** EmailJS
- **Hosting:** Firebase Hosting
